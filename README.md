# Tic Tac Toe Game

A classic two-player Tic Tac Toe game built with HTML, CSS, and JavaScript. Players take turns marking X and O on a 3x3 grid, with the goal of getting three in a row.

## Features

- **Two-Player Gameplay**: Play with a friend on the same device
- **Score Tracking**: Keeps track of wins for X and O players
- **Turn Indicator**: Shows whose turn it is (X or O)
- **Win Detection**: Automatically detects when a player wins
- **Tie Detection**: Detects when the game ends in a draw
- **Winning Animation**: Winning marks scale up and move apart for visual effect
- **Game Controls**: 
  - Reset Game: Reloads the page to start fresh
  - Play Again: Clears the board while keeping scores

## How to Play

1. Players take turns clicking on empty squares
2. X always goes first
3. Get three of your marks in a row (horizontally, vertically, or diagonally) to win
4. If all squares are filled and no one wins, the game is a tie

## Files

- `index.html` - The main HTML structure
- `style.css` - All styling and animations
- `script.js` - Game logic and interactivity

## Setup

1. Download all three files (`index.html`, `style.css`, `script.js`)
2. Place them in the same folder
3. Open `index.html` in any modern web browser

No installation or internet connection required!

## Game Controls

- **Reset Game**: Completely resets the game (reloads the page)
- **Play Again**: Clears the board for a new round (keeps scores)

## Notes

- The game is designed for two human players (no AI)
- Scores persist between rounds when using "Play Again"
